#### Penjelasan
Modul ini bertujuan melatih peserta membangun jaringan dasar menggunakan switch dan router Cisco. Peserta akan merakit topologi, mengonfigurasi perangkat sesuai Addressing Table (termasuk IPv4 dan IPv6), dan memverifikasi konektivitas jaringan. Konfigurasi mencakup pengaturan hostname, passwords, banner motd, dan antarmuka pada Router (R1) dan Switch (S1). Setelah konfigurasi selesai, verifikasi dilakukan dengan tes ping dan perintah IOS seperti show ip route dan show ip interface brief untuk memastikan jaringan berfungsi dengan baik.
